# Shop Man

## Getting started

```
$ npm i
$ npm start
```

-  Setup environment variable

```
$ REACT_APP_BASE_URL=http://localhost:3000
$ REACT_APP_API_URL=http://localhost:5000/api
```

## Feature

-  Login, login with google, facebook.
-  Manage user.
-  Manage products.
-  Manage categories.
-  Search and filter product.
-  Manage product orders.
-  Manage order.
